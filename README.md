# Day Predictor

A simple React project that predicts the day of the week from a given date.  
Built with **React** and bootstrapped using **Create React App**.

---

## Tech Stack
- **Frontend:** React, HTML, CSS, JavaScript
- **Tools:** Create React App, npm
